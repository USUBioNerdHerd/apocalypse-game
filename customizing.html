How to customize the game

Starting point: Finding the right file

The story of the game is in the file called state.json. This file can be found in the main folder (apocalypse-game). This file can be easily opened in TextEdit on a Mac, Notepad in Windows, or a similar text editor of your choosing.

The Introduction

When you open the file, you will see a lot of text, brackets, and colons. Don't panic. This is going to be easy. The top of the file should look something like this:

{"Introduction": "It seems like you've been driving for days, but it has really only been four hours. As a tree detective you have become used to a life of danger and intrigue. Everywhere you go people know your name, Dr. Sammie....", "option 1"


This is the place to put the introduction for your game. Make the game fun by placing a fun background story here. To add your own introduction for the game, replace everything between the "Introduction" : " and ", with your own game introduction. The format should end up looking like this:

{"Introduction": "<YOUR INTRODUCTION HERE>", "option 1":...



"Options"

Information for each decision you make in the game is grouped in the following format throughout state.json:

{"option": " <All of the information related to the choices of the game.>" }



Customizing the Scenarios

From here you should be able to add the information for each scenario by changing the entries for title and summary. Changing these entries will change the title for the scenario and the information given to the user respectively. The responses that the user can choose from can be changed by replacing the text for "first_option" : and "second_option." : Links to scientific articles and popular science magazine articles can be included by replacing the code from the values after "pop-sci" : and "literature" : with the html links for the desired resources. A customized game question will follow the following format.

"option 1": {
"title": " <Add the title for the question here!>",
"summary": "<Put a summary for the question here!>",
"pop-sci": "<Add an html address linking to a popular science article related to your question here!.>",
"literature": "<Add an address linking to a scientific article related to your quesiton here!.>",
"choices": {
"first_option": "<Place a response for the user to choose from here!.>",
"second_option": "<Place a response for the user to choose from here!.>",}},
